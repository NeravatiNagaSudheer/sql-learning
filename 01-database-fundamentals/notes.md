## What I Understood

### Database

A database is a structured storage system used to store and manage data efficiently.

---

### RDBMS

RDBMS stores data inside tables and allows relationships between tables.

Examples:

- PostgreSQL
- MySQL

---

### Table

A table stores related information in rows and columns.

Example:

Users Table

| id | name | email |
|----|------|--------|

---

### Row

A row represents a single record.

Example:

| 1 | Sudheer | sudheer@gmail.com |

---

### Column

A column represents a specific attribute.

Examples:

- id
- name
- email

---

### Primary Key

A Primary Key uniquely identifies a record.

Example:

User ID

1
2
3

No duplicates are allowed.
